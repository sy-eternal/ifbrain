K1中班 财经心理学 1. 理解做选择的意义，形成选择的依据。
2. 理解物品有相同和不同，用来满足不同的需求。
3. 理解需求的层次可划分为必需的和想要的。
账户、储蓄和预算 1. 事情有先后顺序，一些事情必须在另一些事情之前完成。
2. 理解等待的好处，知道锻炼等待的方法。
3. 理解做选择时应有意识地考虑等待的好处，再决定是否要等待。
收入、职业、商业和创业 1. 理解社会分工的意义和种类。
2. 理解社区经济系统分类和作用。
3. 理解不同的工作有不同的报酬。
债权、债务和贷款 1. 理解除了购买外，还有其它获得物品的途径（借用、交易、礼物）。
2. 理解交易的原则、对象和方式。
3. 理解信任是借用的前提，对他人付出要表示感激。
4. 理解分享的好处和方式，学会不愿意分享时候恰当的处理方法。
风险管理与保险 1. 学会选择安全，避开不安全，保护自己。
2. 理解事物间存在因果关系，并且学会接受好结果和坏结果。
3. 理解识别安全行为和不安全行为，学会保护自己。
K2大班 财经心理学 1. 构造在生活必需物品和想要物品间选择的资源分配模型。
2. 了解不同的人有不同的需求，在需要和想要之间做出模拟选择。
3. 了解数字是数量的符号，初步认识货币。
4. 了解满足需要和想要的顺序。
账户、储蓄和预算 1. 掌握数字概念中多和少的概念。
2. 明白较之现在较少所得，储蓄将会获得更多；初步掌握储蓄，利息的概念。
3. 体验模拟银行系统和购买行动。
4. 体验社会中通过从事一份工作或创业赚钱的过程。
收入、职业、商业和创业 1. 了解人有不同的兴趣，我们可以从用不同的方式满足有对应需求的个人。
2. 了解自己的兴趣及在社会中所对应的职业角色，及这些角色如何服务社会。
债权、债务和贷款 1. 了解要与信用良好的人交易，并成为信用良好的借用人。
2. 了解什么样的交易才算公平交易。
3. 理解分享的意义，知道与人相处要索取的同时也要给予。
风险管理与保险 1. 理解选择不同的行为会产生不同的结果, 安全规则会帮助我们做出安全的选择。
2. 知道哪些行为是安全的哪些是不安全的，了解安全演练能保证在意外情况下正确选择安全行为。
3. 判断不同情景安全与否，并选择正确的应对方法。
G1一年级 财经心理学 1. 了解人的必要和想要随时间变化而变化，分清需求的轻重缓急，更快更好的统筹资源，改善自己的生活。
2. 了解需求因人而异的特点，分清每个人的需求都可能不同，更好的处理人际需求和关系。
3. 认识钱币，了解钱的本质含义和计算方法，加强对金额的认识，明白金钱只是一种度量单位。
账户、储蓄和预算 1. 知道储蓄的重要性以及储蓄的好处；能够灵活调整储蓄计划；
2. 能够评估什么是紧急情况并做出正确紧急状态排序；并知道不同紧急状况的不同解决方法
收入、职业、商业和创业 1. 理解从事职业的原因，职业与兴趣的关系。区分服务性职业与产品性职业。在自己兴趣点联想出各种职业，并找到自己最喜欢的职业；
2. 掌握社区内部经济关系和各个职业需要完成的工作内容；
3. 制定出一套完整的商业计划表；
4. 掌握了解客户需求的方法。学会了解产品优势的方法。了解营销手段，例如广告、沟通技巧等。
债权、债务和贷款 1. 理解借出与借入的概念，理解无论借出还是借入的规则，都需要守信用。计算借出与借入的经济价值，识别借东西给他人的风险有多大，并判断自己能不能接受和愿不愿意冒风险；
2. 明白公平交易的概念。理解交易的规则，即等价交换原则。理解不公平交易产生的原因以及交换物品的价值和价格可以有差别。
3. 明白送礼物给别人的概念。 理解送礼物的规则。理解送他人礼物虽然是付出某样物品或服务，但是得到的价值是某种情感的表达形式。
风险管理与保险 1. 了解什么是风险。了解选择承担风险会导致好或不好的结果。辨别风险的大小。选择承担哪些风险是明智的、哪些是不明智的；
2. 了解什么是消极风险。明确生活中不应该冒险（消极风险）的情况；
3. 了解什么是积极风险。明确生活中应该冒险（积极风险）的情况。分析、了解不愿意冒积极风险的原因，以及克服这些困难的方法；
4. 了解风险是不可避免的。知道积极风险也可能产生坏结果；明白小心考量是否承担风险不是逃避风险。
G2二年级 财经心理学 1. 我们可以制定目标帮助我们创造我们想要的生活方式；
2. 我们可以选择我们想要的生活方式；
3. 有的目标比其他目标对我们更重要；
4. 我们对金钱的看法影响我们所做的决定。
账户、储蓄和预算 1. 在某些情况下，最好等待；在某些情况下最好立刻行动；
2. 电子银行和金融机构帮助我们增值储蓄；
3. 制定储蓄计划以提醒自己进行储蓄；
4. 电子银行和金融机构可使我们的储蓄增值。
收入、职业、商业和创业 根据自己的强项选择专业或业务。
债权、债务和贷款 1. 如果我们借了一样东西，我们就应该把它还回去；
2. 我们需要了解他们需要的和想要的，才能和他做公平交易；
3. 我们需要懂得别人的需求，给予有用的礼物。
风险管理与保险 1. 行动前的思考能够帮助我们降低风险；
2. 我们可以通过事情发展的自然顺序或逻辑顺序来谨慎地做出选择；
G3三年级 财经心理学 1. 学生将对他们的选择做出影响和结果预测。然后完成流程图，以系统性地说明做决定的过程；
2. 了解并区分看待金钱的不同心态。批判性思考自己看待金钱的态度；
3. 了解“欲求”和“需求”的定义，确定满足各种“欲求”和“需求”的优先顺序；
4. 明白制定计划对实现目标的重要性及计划过程。明确目标和梦想、短期目标和长期目标之间的区别。
账户、储蓄和预算 1. 学习如何做出花钱和财务管理决定；
2. 通过游戏练习如何做出花钱决定，制定预算。角色扮演家庭预算表，以练习如何做出花钱决定，制定预算；
3. 通过班级商店和赚到的代币，模仿使用银行系统并进行购物练习。
收入、职业、商业和创业 1. 通过亲身体验了解在制作产品和成为社会成员过程中涉及的许多因素；
2. 了解并区分总工资和净工资。然后，确定影响实得工资的支出项目及原因；
3. 根据选择的工作或职业研究并确定预期工资或薪水、总工资和净工资、收入和支出。
债权、债务和贷款 1. 研究购物时使用的不同付款方式：现金、信用卡、支票和借记卡；
2. 通过课堂讨论和可视性T型图探讨信用评级和信贷额度；
3. 用卡账单探讨信贷形式（信用卡、抵押贷款和其他贷款）。
风险管理与保险 1. 通过做游戏学习和讨论风险和预防理念；
2. 学习身份盗用概念及其后果。讨论并学习风险和预防理念；
3. 举例说明日常生活面临的风险产生积极和消极后果的可能性。
G4四年级 财经心理学 1. 辨认好决定和坏决定，明确它们背后的刺激因素。明确有助于未来做出好决定的刺激因素；
2. 明确决策过程的各个步骤，以便做出符合未来目标的选择；
3. 学习如何制定包括这五个方面，即明确性、可衡量性、可达成性、相关性和时限性的目标；
辨认内部和外部动力及刺激因素。明确推动他们内在和外在刺激因素是什么。
账户、储蓄和预算 1. 学习投资来研究时间和金钱之间的关系及时间对金钱的影响；
2. 确定各种时间影响钱生钱的方式，通过实践练习来学习这个概念；
3. 学习投资组合，解释不同投资带来的好处。然后利用实践练习中获得的知识创建一个投资组合。
收入、职业、商业和创业 1. 研究吸引他们的职业或业务。了解从事这些职业或业务的适当步骤；
2. 考虑目前的学习和工作习惯，以确定他们是更适合传统的职业发展道路，还是更适合作为非传统的创业者；
3. 学习区分各种类型的税收以及政府如何征税。
债权、债务和贷款 1. 学习有关条款和贷款条件。然后使用自己的知识面对一些贷款情况做出决定；
2. 吸引学生比较成绩和信用，从而认识到拥有良好的信用将为他们的未来打开一扇大门；
3. 学习负责任地使用信用卡，并设计还款计划。
风险管理与保险 1. 学习保险概念；
2. 通过实践练习举例说明保险概念，展示购买保险的决定如何影响我目前拥有的资金总额；
3. 了解风险类型。
G5五年级 财经心理学 1. 理解什么因素在影响着我们的决定；
2. 理解很多因素可能会影响我们决定；
3. 理解享受现在的生活和为将来储蓄同样重要。
账户、储蓄和预算 1. 理解尽早开始储蓄习惯很重要；
2. 理解满足我们生活各种不同的需要有各种不同的方法。
收入、职业、商业和创业 1. 理解不同职业对应不同的生活方式；
2. 理解为不同的职业或工作需要做不同的准备。
债权、债务和贷款 1. 理解合同告诉我们买方和卖方在利益和风险方面的立场；
2. 理解贷款和其他金融条款所用到的专业词汇。
风险管理与保险 理解比较保险成本和收益的方法。
G6六年级 财经心理学 1. 学习情感与金钱是如何相互关联的。逐步理解情感与金钱之间如何相互影响；
2. 明确并认识一生中激励和影响他们财务状况的各种因素，并培养掌握财务知识的渴望。
3. 确定人际关系与金钱中的主要影响因素。
账户、储蓄和预算 学习制定预算，了解预算是什么？如何使用预算保持财务状况？
收入、职业、商业和创业 1. 创作两篇小故事：一篇描述一个有工作的人；另一篇描述一个有职业的人。这两篇故事应当清楚地描写出工作与职业之间的差别。
2. 深入了解，如何成功改变职业和保持财政稳健，掌握财务知识。
3. 学生将辨认出正在萎缩或增长的行业市场。这些信息将帮助他们未来选择有增长潜力的职业道路。
4. 计算投资回报，接受高等教育能证明持续教育是对未来财务独立做投资。
债权、债务和贷款 1. 完成贷款购物成本和投资评估，与还利息相比，认识赚利息的好处。
2. 认识信贷；
3. 研究不偿还债务造成哪些后果，建立良好的信用带来哪些回报。通过这项研究，他们将准备为决定不偿还债务的朋友写一封劝导信。
4. 每名学生指定一个国家研究对象，研究该国社区目前使用的小额贷款。通过研究指定国家提供的各种小额贷款，学生将制作一本小册子，试图说服投资者为该国民众提供小额贷款。
风险管理与保险 1. 普遍理解保险的含义。
2. 确定需要学习哪些我们需要购买的保险种类，以及复习各类保险。
3. 概述保险索赔过程，以帮助学生了解保险业和索赔过程。
